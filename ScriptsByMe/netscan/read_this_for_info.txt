ARP (Address Resolution Protocol) is used to map IP addresses to MAC addresses within a local network. 
Since we donâ€™t know the MAC addresses of the target devices, we send ARP requests to the broadcast MAC address (ff:ff:ff:ff:ff:ff) asking:
"Who has this IP? Tell me your MAC."
We use Scapyâ€™s srp() function to send and receive these packets at Layer 2. If a host responds, we extract its IP and MAC address and store them in a dictionary.
